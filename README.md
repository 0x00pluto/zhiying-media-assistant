# 智赢媒体助手

在小红书网页上采集笔记、博主、评论，一键下载素材，同步到飞书多维表格，或导出 CSV。

**当前版本**：v0.3.2 · [更新记录](CHANGELOG.md)

---

## 能做什么

| 场景 | 你可以 |
|------|--------|
| **笔记详情** | 下载图片/视频、复制笔记信息、同步飞书、导出评论 |
| **发现 / 搜索 / 博主主页** | 采集当前页面笔记列表，批量拉取详情 |
| **搜索页（用户 Tab）** | 按关键词批量采集博主 |
| **侧边栏批量任务** | 笔记 / 博主 / 评论批量采集，完成后导出 CSV 或同步飞书 |

所有操作都由你主动点击触发，只读取页面上的**公开**内容。

## 安装与打开

1. 在 Chrome / Edge 中安装扩展（应用商店或离线包）。
2. 打开 [小红书网页版](https://www.xiaohongshu.com)。
3. 点击浏览器工具栏中的扩展图标，打开**侧边栏**（快捷键 `Alt + C`）。
4. 建议将网站访问权限设为「**在所有网站上**」，避免部分页面功能受限。

## 快速上手

### 第一步：配置飞书（如需同步）

飞书同步不是必须的。只有要把数据写入多维表格时才需要配置。

1. 在侧边栏右上角点 **设置**（齿轮图标），或右键扩展图标 → **选项**。
2. 进入 **飞书同步**，填写飞书自建应用的 **App ID** 与 **App Secret**。
3. 在飞书开放平台为应用开通**多维表格**读写权限。
4. 保存。

### 第二步：去小红书对应页面

侧边栏首页会提示你在哪类页面会出现采集按钮：

- **发现页** — 采集本页笔记
- **搜索页** — 采集本页笔记；切换到「用户」Tab 可关键词采集博主
- **博主主页** — 采集本页笔记
- **笔记详情**（弹层或独立页）— 下载、复制、同步、导出评论

### 第三步：采集并导出 / 同步

- 单条笔记：在详情页工具栏直接操作。
- 批量数据：页面按钮会把链接导入侧边栏，在侧边栏点「开始采集」，完成后 **导出 CSV** 或 **同步飞书**。

## 使用说明

### 笔记详情页

打开任意笔记后，页面顶部会出现工具栏：

| 按钮 | 作用 |
|------|------|
| **下载图片 / 下载视频** | 将笔记素材保存到本地，按博主昵称与笔记信息分文件夹 |
| **复制笔记信息** | 以表格形式复制到剪贴板，可直接粘贴到 Excel |
| **同步飞书** | 写入你配置的多维表格；首次需选择目标表，之后可一键同步 |
| **导出评论** | 跳转到侧边栏评论采集，自动填入当前笔记链接 |

> **提示**：笔记详情数据最完整。若视频链接为空，请刷新页面后重试，并确保笔记 URL 带有 `xsec_token` 参数。

### 采集本页笔记

在发现页、搜索页（笔记 Tab）、博主主页点击 **采集本页笔记**：

1. 弹窗显示当前页面已识别的笔记数量。
2. 可 **复制链接**，或 **开始采集** 跳转到侧边栏。
3. 在侧边栏设置采集条数，点 **开始采集**。
4. 完成后可预览数据、**导出 CSV**、**同步飞书**。

> **注意**：批量笔记会调用小红书详情接口，请求过快可能触发平台风控。可在设置 → **采集设置** 中控制是否开启批量采集，并适当放慢节奏、减少条数。

### 关键词博主采集

1. 在小红书 **搜索页** 输入关键词，切换到 **用户** Tab。
2. 点击 **关键词博主采集**，侧边栏自动填入关键词。
3. 设置数量上限，**开始采集**。
4. 完成后 **导出 CSV**、**复制**，或 **同步飞书**。

也可在侧边栏手动选择「博主链接」模式，每行粘贴一个主页链接。

### 批量采集评论

**方式一**：笔记详情页点 **导出评论**，自动跳转并填入链接。

**方式二**：侧边栏进入评论采集，粘贴笔记链接（每行一条）。

可设置每条笔记的导出数量、是否包含子评论。采集过程中会显示一级评论与子评论进度。

## 飞书同步

### 表格链接格式

须使用完整 URL，例如：

```text
https://你的域名.feishu.cn/base/APP_TOKEN?table=TABLE_ID
```

也支持 `/wiki/` 知识库链接；分享链接不含 `table` 参数时，扩展会尝试自动识别数据表。

### 同步模式

| 模式 | 说明 |
|------|------|
| **合并** | 按笔记 ID / 评论 ID 等主键更新已有记录，不存在则新增 |
| **追加** | 始终新增记录，不覆盖 |

### 其他能力

- **字段选择**：同步前可勾选要写入的列，支持跳过空值。
- **图片附件**：笔记封面、图片可上传为飞书附件；视频列存链接。
- **历史目标**：各入口（详情 / 批量笔记 / 批量评论 / 批量博主）独立记忆表格链接，下拉可逐条删除。
- **表类型校验**：同步前自动检查目标表是笔记表还是评论表，防止写错表。
- **快捷同步**：勾选「下次不再询问」后，详情页可一键同步到上次目标。

## 设置项

在扩展**选项页**可配置：

| 分类 | 内容 |
|------|------|
| **飞书同步** | App ID / Secret、批量写入上限、素材上传大小与并发数 |
| **采集设置** | 是否开启笔记批量采集（含本页采集、侧边栏批量笔记） |
| **小红书字段** | 导出笔记内容时是否移除话题标签 |

## 采集字段

### 笔记

笔记 ID、链接、类型、标题、内容、话题、点赞/收藏/评论/分享量、发布时间、博主信息、封面与图片链接、视频链接与时长等。

### 博主

博主 ID、昵称、小红书号、粉丝数、获赞与收藏、简介、主页链接、搜索关键词等。

### 评论

评论 ID、内容、图片、点赞量、时间、IP、子评论数、用户信息，以及一级评论 / 引用评论的关联字段等。

具体字段可在同步或导出前自定义勾选。

## 常见问题

**侧边栏没有采集按钮？**  
确认已打开小红书网页，并刷新页面。部分 SPA 跳转后需等页面加载完成。

**飞书同步失败？**  
依次检查：扩展是否已重新加载、是否配置 App ID/Secret、表格链接是否完整、目标表字段是否匹配。可在 `chrome://extensions` → Service Worker → Inspect 查看后台报错。

**批量采集条数为 0？**  
先在页面上点「采集本页笔记」导入链接；评论采集需从笔记详情点「导出评论」导入。

**视频链接为空？**  
详情页场景最可靠。列表批量采集时，若列表接口未返回完整视频信息，视频链接可能缺失。

**触发风控（如 300011）？**  
减少批量条数、降低频率，或在设置中暂时关闭笔记批量采集。
